Welcome to the homepage of the Hinge project

The Hinge project consists of...

  • the Hinge end-to-end encrypted chat system API
  • the HingeChat chatting program implementing the Hinge API

The Hinge project is written in Python 2.7 (in the progress of updating to Python 3.4) and started off as a fork of Cryptully to add group chatting.

Now, the Hinge project plans to expand Hinge into a giant secure chat system,
with all the features a regular chat has, but with security and privacy as well as a decent, clean, intuitive user interface,
universal chatting across multiple platforms with no phone number required, VoIP calling, and much more!

Feel free to contribute to our progress: Hinge organization.

You can visit the Hinge or HingeChat documentation on this website.
