Hinge

Hinge is an end-to-end encrypted chat API.

You can find Hinge's building documentation right here.

But, the internal documentation (which is what you are probably here for), is split up into multiple parts.

  • you'll probably want to start here at Constants to see Hinge's commands and protocol information
  • then, you may want to learn what a Message is
  • of course, you'll want to learn how Hinge's networking system works
  • the TURN server documentation would also be interesting
  • WIP client handshake documentation... this is where it gets confusing (even for me)
