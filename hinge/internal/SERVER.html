TURN Server

The Hinge API uses a builtin central TURN server, which every client connects to via the builtin sock module.
They do not use the sock module directly, they use the Client to handle message sending and receiving. The server can see every client and handles commands through the HingeClient class.
For example, when a client connects to another client, it will send the HELO command, and the HingeClient will pass it on to the other client with the Client as the messenger.
The Client uses the Message module to pack messages, and HingeClient will read it. For more information about commands, see the utils documentation.
